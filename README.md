I once participated in a programming course where we learned data analysis and machine learning. That is why a strong statistical and analytical foundation and awareness of what mathematical operations we perform and how they influence the interpretation of data and results is so important. Writing commands without understanding their meaning will never bring the expected effect because a project begins with thinking, not coding.

Additionally, this approach allows safe experimentation, testing solutions, modifying configurations and introduced changes without the risk of damaging the main system. Currently, I use two virtual machines: one acts as a domain controller, and the other is used for analytical projects, which allows me to work in an environment similar to corporate architectures and develop skills important in teamwork and system-based thinking.
